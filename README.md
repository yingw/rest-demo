## Backend API
- To Create a resource : HTTP POST should be used
- To Retrieve a resource : HTTP GET should be used
- To Update a resource : HTTP PUT should be used
- To Delete a resource : HTTP DELETE should be used

So: 
- Get: `/api/users`
- Get: `/api/users/{id}`
- Put: `/api/users/{id}`
- Post: `/api/users`
- Delete: `/api/users/{id}`
