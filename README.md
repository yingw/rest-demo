# RestTemplate demo

see: http://websystique.com/springmvc/spring-mvc-4-restful-web-services-crud-example-resttemplate/

## Backend API

- Get: `/api/users`
- Get: `/api/users/{id}`
- Put: `/api/users/{id}`
- Post: `/api/users/{id}`
- Delete: `/api/users/{id}`

## Frontend API

- Get: `/users`
- Get: `/users/{id}`
- Get: `/users/{id}/modify`
- Get: `/users/{id}/create`
- Put: `/users/{id}`
- Post: `/users/{id}`
- Delete: `/users/{id}`
